---
description: Build the application for production
---

1. Ensure all environment variables are set in `.env`
2. Run the build command
// turbo
```
npm run build
```
3. Preview the build locally to ensure everything works
```
npm run preview
```
4. Deploy the contents of the `dist` folder to your production hosting provider (e.g., Vercel, Netlify, or a VPS).
